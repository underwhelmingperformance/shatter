# `shatter`

Generate GIFs transitioning from one image to another.

![Tasty ramen transitioning to tastier beer](./ramenbeer.gif)

## Usage

```bash
shatter --duration-seconds 10 --fps 60 Emoji_u1f318.svg.png Emoji_u2600.svg moonsun.gif
```

See `shatter --help` for more options.
